## 採点実感・論文の書き方 総合まとめ

### 法務省 採点実感から抽出した「よくある間違い」

#### 全科目共通の問題点

1. **規範を明示しない**
   - 「〜だから違法である」と結論だけ書く答案が非常に多い
   - 正しくは：規範（判断基準）を示す → 事実を当てはめる → 結論を述べる

2. **条文を引かない**
   - 「信義則に反する」「権利濫用である」と書くだけで、何条の話か不明
   - 条文番号を必ず明記すること。条文の文言から出発する姿勢が重要

3. **問題文の事実を使わない**
   - 問題文に書かれた具体的事実を引用せず、抽象的な議論に終始する答案
   - あてはめでは「本件では〜という事情があるから」と具体的事実を摘示すべき

4. **論点主義に陥る**
   - 論点を次々と書くだけで、事案全体の法的構成ができていない
   - 体系的思考：請求 → 請求権の根拠 → 要件充足 → 効果 → 抗弁

5. **答案の分量配分が不適切**
   - 配点の低い論点に大量に書き、主要論点が手薄になる
   - 出題趣旨を読み、何が問われているかを正確に把握すること

#### 憲法固有の問題点
- 違憲審査基準の選択理由を書かない答案が多い
- 「厳格審査基準による」とだけ書いて、なぜその基準を選んだかの理由がない
- 人権の性質、規制の態様から基準選択の理由を論じること

#### 民法固有の問題点
- 民法改正後の条文を正確に引用できていない
- 契約不適合責任を旧法の瑕疵担保責任の枠組みで書く答案がまだある
- 当事者の主張を整理してから法的検討に入ること

#### 刑法固有の問題点
- 構成要件該当性→違法性→責任の体系的検討ができていない
- 共犯論で「共謀」の認定が雑。具体的事実から共謀の存否を論じること
- 罪数処理を忘れる答案が多い

#### 民事訴訟法固有の問題点
- 条文の文言解釈から出発しない答案が目立つ
- 手続法であることを意識し、具体的な手続の流れの中で論じること

#### 刑事訴訟法固有の問題点
- 強制処分と任意処分の区別基準を正確に論じること
- 令状主義の趣旨から出発して要件を検討する姿勢が重要

### 論文答案の基本パターン

#### 請求権構成（民事系）
```
1. Xの請求の法的根拠
   → 〇〇条に基づく△△請求権

2. 要件の検討
   → 要件①：〜（充足/不充足の理由）
   → 要件②：〜
   → 要件③：〜（ここが主要論点）

3. 抗弁の検討
   → Yの主張：〇〇条に基づく抗弁
   → 抗弁の当否

4. 結論
```

#### 犯罪成立構成（刑事系）
```
1. 甲の罪責
   (1) 〇〇罪（△△条）の成否
       ア 構成要件該当性
         (ア) 客観的構成要件
         (イ) 主観的構成要件（故意）
       イ 違法性阻却事由の有無
       ウ 責任阻却事由の有無
   (2) □□罪との関係（罪数処理）

2. 乙の罪責
   (1) 共同正犯の成否
       ア 共謀の有無
       イ 共謀に基づく実行の有無
```

#### 憲法の答案構成
```
1. 〇〇の自由（△条）の制約
   (1) 保護範囲：〇〇は△条で保障されるか
   (2) 制約の有無

2. 違憲審査基準の設定
   (1) 権利の性質
   (2) 規制の態様
   (3) 基準の選択と理由

3. あてはめ
   (1) 目的の検討
   (2) 手段の検討
   (3) 目的と手段の関連性

4. 結論
```

### 高評価を得るためのコツ

1. **出題趣旨を意識する**：何が問われているかを正確に把握。枝葉末節に時間をかけない

2. **条文から出発する**：すべての法的議論は条文の文言解釈から始まる

3. **事実を拾う**：問題文の事実には全て意味がある。使わなかった事実がないか最後に確認

4. **反対説にも言及する**：判例・通説で書いた上で、「もっとも、〇〇説によれば〜」と付記すると加点要素

5. **分量配分を意識する**：主要論点7割、その他3割が目安

6. **結論は明確に**：「以上より、甲の請求は認められる」と断言する。曖昧な結論は減点

7. **ナンバリングを使う**：1. (1) ア (ア) の階層構造で読みやすくする
